# Pinkesizer

## Description
The Synhesizer is based on:
- MIDI input 
- A waveform generator
- A filter
- Ailtered output 

## Waveform Types 
- Sine
- Sawtooth
- Square
- Triangular
- Two more customized waveforms

## Low Pass Filter Parameters
- Cut-off Frequency
- Quality factor
